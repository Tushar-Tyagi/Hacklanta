# Platform-Specific Notes
# =======================
#
# macOS (Apple Silicon - M1/M2/M3):
#   - Install with: pip install --no-cache-dir -r requirements.txt
#   - Some packages may require Rosetta2: softwareupdate --install-rosetta
#   - For tensorflow, consider using tensorflow-macos:
#     pip install tensorflow-macos tensorflow-metal
#
# macOS (Intel):
#   - Standard pip installation should work out of the box
#   - May need to install OpenCV with brew: brew install opencv
#
# Linux (Ubuntu/Debian):
#   - Install system dependencies first:
#     sudo apt-get update
#     sudo apt-get install -y libgl1-mesa-glx libglib2.0-0 libsm6 libxext6 libxrender-dev libgomp1
#   - For face recognition, you may need: sudo apt-get install -y dlib libface-recognition0
#
# Windows:
#   - Use Visual C++ Redistributable for OpenCV
#   - Download from: https://aka.ms/vs/17/release/vc_redist.x64.exe
#
# Headless Servers (Linux/WSL):
#   - Use opencv-python-headless instead of opencv-python
#   - Install: sudo apt-get install -y libgl1-mesa-glx libglib2.0-0
#
# GPU Acceleration (NVIDIA CUDA):
#   - Install CUDA Toolkit 11.8+ and cuDNN
#   - Then install PyTorch with CUDA:
#     pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
#   - Verify with: python -c "import torch; print(torch.cuda.is_available())"
#
# Troubleshooting
# ===============
# If build errors occur:
#   1. Update pip: pip install --upgrade pip setuptools wheel
#   2. Install build tools: pip install cmake
#   3. For DeepFace issues: pip install dlib --precomputed
#
# If librosa fails to load audio files:
#   - Install ffmpeg: brew install ffmpeg (macOS)
#   - sudo apt-get install ffmpeg (Linux)
#   - pip install ffmpeg-python
#
